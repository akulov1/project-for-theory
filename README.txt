Для запуска программы необходимо открыть Program.exe
По умолчанию количество генерируемых вариантов = 1.
После выбора нужных заданий нажмите кнопку "Создать документ", после которой откроется маленькое окно, в котором нужно будет нажать "ОК".
После нажатия кнопки "ОК" программа автоматически закроется через 1 секунду, файлы "задания.doc" и "ответы.doc" обновятся автоматически.
